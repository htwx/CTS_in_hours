## 一 准备编程

编程就是让计算机做你想让它做的事.

编程语言是工具,就像画笔,应该拿上手找块空白就可以用.

为了编写第一个TypeScript程序, 最简单的方法是安装Node. (待续: node推荐版本, 如何运行tsc.js等)

本文的代码足够简单,集成开发环境的用处不大,任何文本编辑器都可以(推荐工具待定.写本文时用的是Visual Studio Code).

安装Node后, 打开命令行窗口,运行`node -v`, 不报错即为成功,可以继续. (未完:常见问题与解决)
